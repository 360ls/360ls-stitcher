stitcher.client module¶

This module is responsible for setting up the client-socket pairing for streaming of video output. It is simply a script and does not contain any classes or function declarations.

stitcher.client.main()¶

The main function for the client.py script.

stitcher.configure module¶

This module is responsible for taking in user input from the terminal and turning it into a configuration profile.

stitcher.configure.main()¶

Calls the main parse method.

stitcher.configure.parse()¶

Parses user input to pass to the configuration profile.
